# Магазин

### Описание

Покупка разных предметов, которые будут храниться в [инвенторе](/KEKWaitBot/docs/gold/inventory)


## Использование комманды
 **`!shop`**

 Просмотр товаров в магазине.

## Покупка предметов
 **`!shop *название предмета* *amount*`**

 - `amount` - количество 

 Покупка предметов в магазине.
