# Баланс

:::danger Внимание

Баланс во время стрима отключён

:::

### Описание

Узнать текущее количество золота и место в топе.

 **`!баланс`**

 

:::note

Можно посмотреть баланс другого зрителя:

**`!баланс User`**
- `user` - ник зрителя

:::
