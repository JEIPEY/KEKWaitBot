# Слоты

:::danger Внимание

Слоты во время стрима отключены

:::

### Описание

Сыграйте в слоты, что бы увеличить свою ставку в несколько раз!



 **`!слоты amount`**

- `amount` - количество 

:::note

В качестве `amount` можно использовать **all** или **всё**, что бы поставить всё золото на балансе.

:::
