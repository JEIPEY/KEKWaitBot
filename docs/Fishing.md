# Рыбалка

### Описание

Запуская удочку **в воду** есть шан на поимку рыбы, предметов или мусора, за которые можно получить Золото.

:::warning Внимание

Рыбалка во время стрима отключена

:::

## Использование комманды
 **`KEKBaited`**

 

:::tip

**KEKBaited** можно использовать в любой части сообщения

:::

  ## Дополнение

### Пузыри


Раз 1-2 часа появляются пузыри. В течении одной минуты после появления пузерей можно поймать особых рыб, с повышеной стоимостью. Так-же пузырьки позволяют ловить рыбу во время стрима.
