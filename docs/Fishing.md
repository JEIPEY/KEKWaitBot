# Рыбалка

### Описание

Запуская удочку **в воду** есть шан на поимку рыбы, предметов или мусора, за которые можно получить Золото.

:::warning Важно

Рыбалка во время стрима отключена

:::

## Использование комманды
 **`KEKBaited`**

 

:::tip Совет

**KEKBaited** можно использовать в любой части сообщения

:::

## Предметы

:::caution Внимание

Предметы используються сразу после покупки и по одному типу.

Приоритет использования предметов **Магнит** > **Наживка**

:::

### Наживка

Предмет который можно купить в [магазине](/KEKWaitBot/docs/gold/shop).

посмотреть количество наживки можно в [инвентаре](/KEKWaitBot/docs/info/inventory).

:::tip Убирает шанс поймать мусор.

:::


:::info Наживка имеет шанс не потратиться.

:::


:::info Если наживка закончилась, рыбу всё ещё можно будет поймать, но с меньшим шансом.

:::


### Магнит

Предмет который можно купить в [магазине](/KEKWaitBot/docs/gold/shop).

Посмотреть количество магнитов можно в [инвентаре](/KEKWaitBot/docs/info/inventory).

:::tip Вместо рыбы ловит с повышеным шансом сокровище и с меньшим мусор.

:::


:::info Если магниты закончились, сокровище всё ещё можно будет поймать, но с меньшим шансом.

:::

  ## Дополнение

### Пузыри


Раз 1-2 часа появляются пузыри. В течении двух минут после появления пузерей можно поймать особых рыб, с повышеной стоимостью. Так-же пузырьки позволяют ловить рыбу во время стрима.


:::caution Предметы влияющие на шансы, не работают с пузырьками.

:::
