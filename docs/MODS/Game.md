# Название игры

## Описание

Узнать текущее название игры

**`!game`**


<div>

| Global cooldown | 10 seconds⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀|
|:----------------|:----------------------|
| User cooldown   | 0 seconds            |
| Mod only        | No                    |
| Sub only        | No                    |
| Vip only        | No                    |
| Другие варианты комманды | !game !игра !гейм|

</div>

## Модерация

Изменить текущее название игры.


**`!game *название игры*`**

:::note При ошибке напсиания выдаст ошибку
:::
