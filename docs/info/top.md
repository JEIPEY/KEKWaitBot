# Топ
### Описание

Узнать топ зрителей по золоту.


 **`!топ`**

## Пример использования
![Docs Version Dropdown](./img/top.png)

  <div>

| Global cooldown | 20 seconds⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀|
|:----------------|:----------------------|
| User cooldown   | 0 seconds            |
| Mod only        | No                    |
| Sub only        | No                    |
| Vip only        | No                    |
| Другие варианты комманды        | !top              |
  </div>
