# Кубик и монетка

## Кубик

### Описание

Кинуть кубик с любым значением от 1 до 100.

## Использование комманды

 **`!roll amount`**

 - **amount** число от 1 до 100

## Пример использования


  <div>

| Global cooldown | 10 seconds⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀|
|:----------------|:----------------------|
| User cooldown   | 0 seconds            |
| Mod only        | No                    |
| Sub only        | No                    |
| Vip only        | No                    |
| Другие варианты комманды        | !dice !кубик              |
  </div>

## Дополнение

### Монетка
